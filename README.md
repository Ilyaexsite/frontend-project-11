## Клонирование репозитория
```bash
git clone https://github.com/Ilyaexsite/frontend-project-11.git
cd frontend-project-11
```

# Установка зависимостей
```bash
npm install
```

# Запуск в режиме разработки
```bash
npm run dev
```

# Сборка для production
```bash
npm run build
```

# Превью собранного приложения
```bash
npm run preview
```

# Проверка кода линтером
```bash
npm run lint
```

# Автоисправление ошибок линтера
```bash
npm run lint:fix
```

# Запуск тестов
```bash
npm run test
```

# Запуск тестов с покрытием
```bash
npm run coverage
```
